YelpCamp

Source Code

Live Demo

This application was my first full-stack web application. This is a toy replica of Yelp, but for campsites, where the main item being "marketed" is a campsite.

Some of the interesting things I learned was RESTful routing, Authentication and Authorization. Mapbox enabled me to use Geocoding in the application and I also implemented a Fuzzy Search for the campsites. Other popular features of Yelp such as Pricing and Reviews/Comments are also present.

I used HTML, CSS and Bootstrap for the front-end. The server used was Express.js with a MongoDB instance for persistent data. The app itself is hosted on Heroku, on a Node.js environment.


Guess The Password ES2015 Refactor

Source Code

Live Demo

A simple refactor of a pre-existing JavaScript application from ES5 to ES2015 features. The game uses vanilla JavaScript along with HTML and CSS.

The user is asked to guess the "right" password from a list of words. There are only a fixed number of attempts which are decided at random themself ;) and the user is informed how many letters were right from their choice after every attempt.

The ES2015 features I included were:

  • Arrow Functions
  • 'let' and 'const' keywords
  • Array destructuring
  • Template Strings
  • Rest and Spread operators

I tackled this problem as in the following manner. First I had to run local instances of GitLab and Jenkis, which was possible with the help of Docker. GitHub Repository Analyzer

Source Code

Live Demo

Our task was to create a GitHub repository analyzer for Java Projects. We had to use the Understand tool to analyze specifics of the source code such as Entities and References.

The first step in my solution was to retrieve open source repositories from GitHub. Next, I obtained all necessary patch files relevant to the source code and this enabled me to retrieve the consecutive versions of the source code, due to the commit hashes. Then using SciTools Understand, we iteratively created and compared the "meta codebase" for two consecutive versions of the source, allowing to publish the results of comparison onto JSON.

Source Code

Live Demo
